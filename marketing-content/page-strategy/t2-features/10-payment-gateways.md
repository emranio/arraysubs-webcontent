## Content Blocks

### 1. Hero Section
- H1 + subtitle: "Works with any WooCommerce payment gateway — zero lock-in"
- Gateway logos: Stripe, PayPal, Paddle + "all WooCommerce gateways"
- CTA: Download Free

### 2. How Payment Works
- **Any WooCommerce Gateway**: First payment through any active gateway
- **Manual Renewals (Free)**: Renewal invoices sent to customer; they pay at their convenience
- **Automatic Renewals (Pro)**: Auto-charge saved payment methods via tokenized gateways
- Visual: Payment flow diagram

### 3. Supported Gateway Types
| Gateway | First Payment | Manual Renewal | Auto Renewal (Pro) |
|---------|--------------|----------------|-------------------|
| Stripe | ✅ | ✅ | ✅ |
| PayPal | ✅ | ✅ | ✅ (tokenized) |
| Paddle | ✅ | ✅ | ✅ |
| Bank Transfer | ✅ | ✅ | ❌ |
| COD | ✅ | ✅ | ❌ |
| Any WooCommerce | ✅ | ✅ | Depends on tokenization |

### 4. Automatic Payments (Pro Badge)
- Save card on file during first payment
- Auto-charge on renewal due date
- Customer can toggle auto-renew on/off from portal
- Payment retry on failure with configurable schedule
- Visual: Auto-renew toggle in customer portal

### 5. Store API & Block Checkout Compatibility
- Full support for WooCommerce block checkout
- Store API compatible for headless/custom frontends
- Works with both classic and block payment flows

### 6. FAQ

| Question | Answer Focus |
|----------|-------------|
| Can I use any payment gateway? | Yes — any WooCommerce-compatible gateway works for first payment and manual renewals |
| Is Stripe supported? | Yes — with automatic renewals in Pro |
| Is PayPal supported? | Yes — with automatic renewals in Pro (tokenized) |
| Is automatic billing free? | No — auto-charge is Pro; manual invoicing is free |
| Does it work with block checkout? | Yes — full Store API compatibility |
