### 1. Hero Section
- H1 + subtitle: "Modern admin interface for subscription management"
- Screenshot: Subscription list dashboard
- CTA: Download Free

### 2. Subscription List View
- Sortable, filterable table of all subscriptions
- Status filters (Active, Paused, On-Hold, Cancelled, Expired)
- Search by customer, product, or ID
- Bulk actions
- Visual: List view screenshot

### 3. Subscription Detail View
- Complete subscription information
- Status with manual override capability
- Customer info and order history
- Renewal schedule and next payment
- Meta data and custom fields
- Quick actions: Pause, Resume, Cancel, Renew Now
- Visual: Detail view screenshot

### 4. Bulk Operations
- Bulk status changes
- Bulk renewal processing

- Change status with one click
- Process manual renewal
- View related orders
- Navigate to customer profile
- View audit log (Pro)
