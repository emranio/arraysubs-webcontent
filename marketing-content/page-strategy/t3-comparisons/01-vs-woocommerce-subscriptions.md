### 4. Feature Deep-Dives

**4a. Subscriptions & Billing (Tie with edge to ArraySubs)**
- Both handle recurring billing well
- ArraySubs adds two-phase grace periods (WCS doesn't differentiate)
- ArraySubs has setup wizard (WCS doesn't)

**4b. Memberships & Access Control (ArraySubs wins)**
- ArraySubs: Built-in member access, content restriction, role mapping, drip content
- WCS: Requires separate WooCommerce Memberships plugin ($199/yr)
- Total cost with memberships: ArraySubs $0–$X vs WCS $478/yr

**4c. Retention & Churn (ArraySubs wins — WCS has nothing)**
- ArraySubs: Retention flow builder with offers, reasons, analytics
- WCS: Cancel button → gone. No intercept.

**4d. Store Credit (ArraySubs wins — WCS has nothing)**
- ArraySubs Pro: Full store credit system
- WCS: No store credit functionality

**4e. Developer Experience**
- Both: Hooks, filters, REST API
- ArraySubs: Modern React admin, PSR-4 autoloading
- WCS: Traditional WordPress admin
