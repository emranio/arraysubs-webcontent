### 2. What You Can Do with Stripe + ArraySubs
- Accept first subscription payments via Stripe
- Save card on file for automatic renewals (Pro)
- Customer updates payment method from portal
- Handles SCA/3D Secure compliance

### 3. Step-by-Step Setup Guide (HowTo Schema)
1. Install WooCommerce Stripe Gateway
2. Connect Stripe account
3. Enable saved payment methods
4. Install ArraySubs
5. Create subscription product
6. Test checkout

### 4. Free vs Pro with Stripe
- **Free**: Stripe processes initial payment; renewal invoices sent manually
- **Pro**: Stripe auto-charges saved card on renewal date; retry on failure
