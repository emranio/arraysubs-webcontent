## Content Blocks

### 1. PageHero — "Turn WooCommerce into your SaaS billing engine"
### 2. SaaS Billing Model
- Tiered plans (Basic, Pro, Enterprise via variable subscriptions)
- Monthly / Annual billing
- Free trials for onboarding
- Signup fees for setup costs

### 3. Feature-Gated Access
- Use member access to gate features by plan
- Content restriction for documentation tiers
- Download restriction for software versions
- Feature Manager (Pro) for granular entitlements

### 4. Credits & Usage Models
- Store credit (Pro) as credits/tokens system
- Top-up credit products for usage-based billing

### 5. Customer Self-Service
- Portal for plan management
- Upgrade/downgrade through plan changes
- Billing history and invoice access

### 6. Retention for SaaS
- Retention flows catch cancellations
- Offer discounts or downgrades instead
- Churn analytics to identify at-risk users

### 7. Technical Integration
- REST API for external app integration
- Webhooks via Action Scheduler hooks
- Store API compatibility

### 8. Step-by-Step: Build a SaaS Billing System
1. Create tiered subscription products
2. Set up access rules per plan
3. Configure billing and grace periods
4. Enable customer portal
5. Set up retention flows
