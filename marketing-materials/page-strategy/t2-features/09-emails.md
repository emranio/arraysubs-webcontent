## Content Blocks

### 1. Hero Section
- H1 + subtitle: "Every subscription event triggers the right email"
- Screenshot: Email settings screen
- CTA: Download Free

### 2. Email Types Available

| Email | Trigger | Recipient |
|-------|---------|-----------|
| New Subscription | Subscription created | Customer + Admin |
| Subscription Activated | Status → Active | Customer |
| Subscription On-Hold | Status → On-Hold | Customer |
| Subscription Cancelled | Status → Cancelled | Customer + Admin |
| Subscription Expired | Status → Expired | Customer |
| Renewal Reminder | X days before renewal | Customer |
| Renewal Invoice | Invoice generated | Customer |
| Payment Failed | Payment attempt fails | Customer + Admin |
| Expiring Soon | Subscription nearing end | Customer |

### 3. WooCommerce Email Integration
- Uses native WooCommerce email system
- Inherits WooCommerce email templates and styling
- Customizable via WooCommerce → Settings → Emails
- Template overrides in theme
- Visual: WooCommerce email settings showing subscription emails

### 4. Email Customization
- Edit subject lines and preheaders
- Customize body content with template tags
- Enable/disable individual emails
- Separate admin and customer notifications
- Visual: Email template editor

### 5. Renewal Reminders
- Configurable days-before-due reminders
- Help customers prepare for upcoming charges
- Reduce payment failures with early notification
- Link to pay early or update payment method

### 6. FAQ

| Question | Answer Focus |

| Do renewal reminders work? | Yes — configurable days-before-due reminders |
