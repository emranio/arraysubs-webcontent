## Content Blocks

### 1. Hero Section (Pro Badge)
- H1 + subtitle: "The only WooCommerce subscription plugin that shows customers exactly what each plan includes"
- Blue Ocean badge: "Unique to ArraySubs"
- Screenshot: Product page "What's Included" section with toggle/number/text features
- CTA Buttons: Live Demo + Get Pro Free for 6 Months

### 2. Feature Manager Overview
- Define named features (entitlements) per product or per variation
- 3 feature types: Toggle (on/off), Number (limits/quotas), Text (descriptive details)
- Features update dynamically when customers select variations
- Visual: Product edit screen with Feature Manager panel open

### 3. Three Feature Types Deep Dive
- **Toggle Features**: Perfect for on/off entitlements — "Priority Support: ✅", "Custom Domain: ❌"
- **Number Features**: Ideal for limits and quotas — "5 Team Seats", "100 API Calls/month", "50 GB Storage"
- **Text Features**: Best for descriptive plan details — "Standard SLA", "Dedicated Account Manager"
- Visual: Each type with product edit + storefront rendering side by side

### 4. Feature Templates
- Save feature sets as reusable templates
- Apply templates to new products instantly
- Customize per-product after applying
- Maintain consistency across subscription catalog
- Visual: Template save/load workflow

### 5. Storefront Display — "What's Included"
- Automatic "What's Included" section on product pages
- Toggle features show ✅/❌ with comparison view option
- Number and text features display values
- Variable products update dynamically per variation
- Visual: Product page with "What's Included" rendered below product summary

### 6. Customer Portal — "My Features"
- Dedicated My Account page showing all active entitlements
- Two aggregation modes: Per-Subscription or Combined
- Combined mode: toggles use logical OR, numbers sum across subscriptions
- Visual: Customer portal My Features page

### 7. Usage Tracking
- Track consumption of metered entitlements (API calls, downloads, seats)
- Helper functions for recording, resetting, and checking usage
- Customer-facing "Used X of Y" display
- Admin-visible usage data in Feature Log
- Visual: Usage counter display in portal and admin

### 8. Admin Feature Log
- Per-subscription feature entitlement log on the admin subscription detail page
- Shows feature name, type, value, and usage
- Reflects current product configuration
- Visual: Subscription admin page with Feature Log section

### 9. Business Use Cases
- **SaaS**: API limits, team seats, storage quotas per plan tier
- **Memberships**: Premium content access, community features, download tiers
- **Service Businesses**: Support levels, SLA terms, dedicated contacts
- **Online Courses**: Bonus modules, quiz limits, certification eligibility
