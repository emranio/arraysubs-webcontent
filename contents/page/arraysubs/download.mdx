## How to Install

### Step 1 — Install the Plugin

From your WordPress admin:

1. Go to **Plugins → Add New**
2. Search for **ArraySubs**
3. Click **Install Now**, then **Activate**

Alternatively, [download the zip from WordPress.org](https://wordpress.org/plugins/arraysubs/) and upload it via **Plugins → Add New → Upload Plugin**.

### Step 2 — Activate & Run the Setup Wizard

After activation, the **ArraySubs Setup Wizard** launches automatically. It walks you through:

- Connecting WooCommerce
- Choosing your billing model (subscriptions, memberships, or both)
- Configuring your first product

The wizard takes under 3 minutes. Full setup documentation is at [support.arrayhash.com/arraysubs](https://support.arrayhash.com/arraysubs/).

### Step 3 — Create Your First Subscription Product

Once setup is complete, you can create subscription products from **Products → Add New**. Choose a billing interval, set your price, and publish.
